O script a seguir foi desenvolvido para ser executado e receber interações apenas através de uma CLI.

Para compilar e executar o script, siga o seguinte passo a passo:
--> OBS IMPORTANTE: Este passo a passo funcionará apenas em sistemas operacionas baseados em UNIX.

1° Passo - Mova a pasta CommandStackIndustrialRobot (pasta onde se encontra este arquivo) para a pasta Documents do seu sistema operacional.

2° Passo - Abra o terminal (você pode usar o atalho CTRL+ALT+T).

3° Passo - Para compilar o script, digite os seguintes comandos:
    cd Documents/CommandStackIndustrialRobot/
    gcc industrial_robot_commands.c -o industrial_robot_commands


4° Passo - Por fim, execute o script com o seguinte comando:
    ./industrial_robot_commands
